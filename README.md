**Student Record Management System Project**
This project stores the information about a student like his/her name,age,subject and interests.
It gives multiple options to its users that whether they want to**View the profile of student,change the age or name of student,add interests or they want to exit the program.**
It has various functions for performing specific tasks.
Language:Python
